# 🖥️ MODAL VESSEL OS v4.0

> An interactive terminal simulation exploring the Codex Modalis universe.

## 🎮 How to Use

1. Open the site (link below).
2. Watch the boot sequence.
3. Choose your access level (**GUEST** or **ADMIN**).
4. Explore the system commands.

## 🛠️ Tech Stack

- **HTML5** (Structure)
- **CSS3** (Animations, Styling)
- **JavaScript** (Interactivity)

## 📜 Lore

Based on the **Codex Modalis** universe where magic is code, reality is a machine, and the Indifferent weaves webs of evolution.
